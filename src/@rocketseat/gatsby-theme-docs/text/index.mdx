# Welcome to Gatsby WP Themes

**Gatsby WP Themes** are Gatsby themes designed to work with your WordPress content. We built them easy to start with. We also took care to make them customizable.

If you use WordPress but would like to try using it the Jamstack way, our themes are a great option.

## Features

- Ready to use demos, quick start with some content
- Dynamic comments
- Search functionnality
- Mailchimp integration
- Support for forms
- Detailed documentation
- On-line Support
