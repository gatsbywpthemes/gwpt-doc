## General settings

You can override some of the WordPress Theme Options set via WordPress Customizer ([see WordPress Theme Options](/configuration/wp)) in your Gatsby `config.js` file.

```javascript title=config.js
const config = {
  // ...
  addWordPressComments: false,
  addWordPressSearch: true,
  addColorModes: false,
  socialFollowLinks: [
    {
      name: "codepen",
      url: "https://codepen.io/pehaa",
    },
  ],
}
module.exports = config
```

The options set in `config.js` will override your WordPress settings.

If you want to override color settings please check the [Colors Configuration section.](/configuration/colors/#setting-colors-in-js-configuration-file)

> In the development mode, you will have to **restart the development server** to see any changes from the config file.
